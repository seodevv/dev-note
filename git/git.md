## add
+ file 을 staging area 에 포함 시킨다.
```
> git add {fileName}
> git add {fileName1} {fileName2}
> git add .
```
<img src="./gitAdd.png" alt="gitAdd.png" />

## commit
+ staging area 에 있는 file 을 repository 에 저장한다.
```
git commit -m 'message'
```
