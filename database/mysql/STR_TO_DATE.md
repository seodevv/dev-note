# STR_TO_DATE


## property
| Format | desc | example | Format | desc | example |
|--------|------|---------|--------|------|---------|
| %Y | 4자리 년도 | 2023 | %y | 2자리 년도 | 23 |
| %M | 영문 월 | %m | 2자리 월 |
| %W | 영문 일 | %d | 2글자 일 |
| %H | 24시간 | %h | 12시간 |


## syntax
``` 
mysql> select STR_TO_DATE('2013-09-05T10:10:02Z','%Y-%m-%dT%TZ');
+----------------------------------------------------+
| STR_TO_DATE('2013-09-05T10:10:02Z','%Y-%m-%dT%TZ') |
+----------------------------------------------------+
| 2013-09-05 10:10:02                                |
+----------------------------------------------------+
1 row in set (0.00 sec)
```
